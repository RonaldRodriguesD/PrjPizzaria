Sobre o Projeto

Projeto Web desenvolvido para disciplina de Desenvolvimento Web III da Faculdade de Tecnologia - FATEC Zona Sul.

Genis Ferreira e Ronald Rodrigues

Desenvolvemos um site para Pizzaria onde o cliente poderá visualizar o cardápio e montar sua pizza escolhendo entre os sabores disponíveis, tamanho, borda e ingredientes.

O sistema calculará automaticamente os valores e apresentará a descrição das escolhas e valor do pedido.
